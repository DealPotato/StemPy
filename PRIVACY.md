# Privacy

StemPy processes song audio locally. It does not include analytics, telemetry,
advertising, user accounts, or a StemPy-operated cloud service.

## Network Access

StemPy or its launcher can access the network for:

- Python and FFmpeg installation.
- Python package installation and runtime repair.
- Loading the Audio Separator model catalog.
- Downloading user-selected AI models.

Song audio is not intentionally uploaded by StemPy.

## Local Data

StemPy stores settings, custom presets, downloaded models, temporary files,
backups, and diagnostic logs on the local computer. Logs can contain song names
and absolute filesystem paths. Review and redact logs before posting them
publicly if those paths contain personal information.
